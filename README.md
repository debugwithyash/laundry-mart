# LaundryMart Website

LaundryMart is a simple laundry service website created using HTML.
____
This project is designed for users to view laundry services, pricing, and contact details in a clean and attractive layout.
🚀 Live Demo
👉 https://debugwithyash.github.io/laundry-mart/
_____


## Features

* Responsive Header Section
* Service Details Section
* Pricing Section
* Simple and Clean Layout

## Technologies Used

* HTML

## Project Purpose

The main purpose of this project is to practice HTML structure while creating a real-world service-based website.

## Future Improvements

* Add responsive design for mobile devices
* Add online booking form
* Add service images
* Add customer reviews section
* Add CSS styling in the future
* Add JavaScript for interactivity
